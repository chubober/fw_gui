The search form allows for selecting different features and combining them with each other. AND and OR logical operators are available. Every search card refers to one search query, and after queries are completed, they are connected via logical OR. Within a card, fields are connected via logical AND (features within one field are connected via OR). It is possible to add more text-type fields if needed (they can be connected either via AND or OR).
